# My dotfiles

This is my current dotfiles repository.

To install these you can run the following command:

```sh
./install.sh
```

## Requirements

For installation, you need `stow` program.

Mentoined programs:
- `niri`
- `alacritty`
- `btop`
- `kitty`
- `fastfetch`
- `qt5ct`
- `qt6ct`
- `zsh`
- `MangoHud`
- `spotify-launcher`
- `kanshi`
- `starship`
- `noctalia-shell`

Also, `eplayerctl` script is required. It is stored in 'bin/eplayerctl' directory and is automatically linked into '~/.local/bin' when using `install.sh` script.
